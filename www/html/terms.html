What data do we collect?

We collect and store your username and nickname and messages you send. We do not store or process cookies, IP addresses or any other information used to uniquely identify a device. We do not store or process any personal data including names, dates of birth, location data or any other personal information.

We understand that privacy is important. Therefore, we don't ask for your contact or personal information. If you need to recover your account, we use TOTP (Time-based one time passwords) to confirm your identity. Make sure to add us to your authenticator app in your account settings!

How long do we keep it for?

We store usernames and nicknames until a user deletes their account. We store message history of a room until the room is disbanded (when all members have left). Messages are only stored so the message history can be recovered when new members join.

What do we do with data?

Usernames are used to identify user accounts. We generate an authentication token which is associated with a user account once the user has logged in. Authentication tokens are reset when a user changes their password. We do not transmit any user information outside of the YTParty API. Passwords are NEVER stored in a plain text format.

How do we store data?

All user data is stored in a PostgreSQL database. Passwords are hashed instead of being stored in plain text. Data is only accessible via the YTParty API using an authorisation token. Tokens for one user cannot be used for another. Some data that users input cannot be retrieved via the API for security purposes, however this data can be erased or modified by deleting an account or modifying the account settings.
