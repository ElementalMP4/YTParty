What data do we collect?

We collect and store your username and nickname and messages you send. We do not store or process IP addresses or cookies. We do not store or process any personal data.

How long do we keep it for?

We store usernames and nicknames until a user deletes their account. We store message history of a room until the room is disbanded (when all members have left).

What do we do with data?

Usernames are used to identify user accounts. We generate an authentication token which is associated with a user account once the user has logged in. Authentication tokens are reset when a user changes their password.

How do we store data?

All user data is stored in our database. Passwords are hashed instead of being stored in plain text. Data is only accessible via the YTParty API.
